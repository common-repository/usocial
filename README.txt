== Description ==

uSocial is a modern, user-friendly Like and Share button builder for your website.

This plugin allows you to:

* add Like and Share buttons to all your site pages in a few clicks;
* add the buttons to <body>, after the h1 heading, and to the product card;
* analyze the button set stats.


### Features:

* can be used only for adding the uSocial widgets to your site;
* allows you to add pop-up windows, the uLike widget, and Social Locker from uSocial to all your site pages.

== Installation ==

Just install from your WordPress "Plugins > Add New" screen and all will be well. Manual installation is very straightforward as well:

1. Upload the zip file and unzip it in the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. You will find 'Usocial' menu in your WordPress admin panel.
